# Gmail-Converstion-View
1)Get conversations of mails</br>
2) New UI that can handle using one hand </br>
3)Read and Send Mails with Attachments

1)Android Application Developments </br>
2)Java Programming  </br>
3)SQL </br>
4)Gmail API</br>
